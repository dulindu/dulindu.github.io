    Ambulance Emergency Patient Administration and Remote Healthcare Monitoring System

    In this project I designed a real-time monitoring IoT system to interpret patient’s vital signs and other related information securely with healthcare experties while the patient’s being delivered by the ambulance.
    This application was designed and localy hosted by using Flask web frame work and I used 4 sensors of my android mobile phone as temporary sensors to represent vital signs. Sensor values were sent to the dash board real time with a help of a third party app run on the android. Additionaly, sms alerts, gamil alerts and otp security features were integrated.

    Libraries and APIs used in this apllication,

  • Flask Web Framework
  • Xlrd, Xlutils - Excel Data handelling
  • Socket - Transfer Sensory Data
  • Apex Chart - Dashboard real-time graphs
  • Way2SMS - SMS alerts

    • Feature Extraction and Classication of EEG Signals for Brain Computer Interface (BCI)

    In this project I developed an algorithm to extract band power features of EEG signals taken from Emotiv Insight EEG headset to build a brain computer interface which can classify different controlls with the help of different mental status.
    The algorithm predicts the mental states based upon a formula made by band power components of several EEE waves. Baisically, relaxation, concentration, and eye blinking states can be captured on this formula which can lead to several commands. Algorithm is tuned up at any new user to update the thresholds which deviate mental status.
    I tested the algorithm for a two way light switching use case to measure the accuracy. Accuracy is about 87% according to the results recorded for 3 persons.
